<h3><b>Step 1. Installation Manual</b></h3>
<hr>
<p><b>I.</b>	Download Script, then extract it on your computer</p>
<p><b>II.</b>	 Upload the “Affiliate Pro v--.zip” file to your public_html or localhost folder.</p>
<p><b>III.</b>	Extract the zip file.</p>
<p><b>IV.</b>	Create a new database then link the database with a user.</p>
<p><b>V.</b>	Go to phpmyadmin click on the newly created database then import the “Affiliate Pro v-.sql” it’s located among the files extracted on your computer @ (I).Now Db is set.</p>
<p><b>VI.</b>	Let’s set up the config.php, navigate to public_html/controller/config.php</p>
<p><b>VI.</b>	VII.	set up config.php[v 1.1] , navigate to /install/ copy  “config.tpl” to /controller/ resulting to /controller/config.php [v 1.1]</p>
<p><b>VII.</b>	Open it; fill in the corresponding fields (Line 30-41) save it.[v 1.0]</p>
<p><b>VIII.</b>	Congratulation, Setup is now complete.</p>
<p><b>IX.</b>	For SSL add s to http on line 20 of your config.php</p>
<p><b>X.</b>	For SSL force all http to https uncomment line 2-4 of your .htaccess file, location public_html/.htaccess</p>
<p><b>XI.</b>	Video: Installation Steps https://youtu.be/diCUlhnK08s</p>
